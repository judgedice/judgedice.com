# Card

Restrained content container — paper-raised, hairline border, near-sharp corners, no heavy shadow. `framed` drops the border for a single 2px ink rule along the top (editorial index-card look). `hover` adds a lift for clickable cards.

```jsx
<Card>
  <SectionLabel color="muted">Marketing Technology Architecture</SectionLabel>
  <p>End-to-end systems that hold together under real conditions.</p>
</Card>
<Card framed hover>…</Card>
```

Props: `framed`, `hover`.
