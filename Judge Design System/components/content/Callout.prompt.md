# Callout

The dry aside / pull-quote — a left vermilion rule and italic serif. Where the free-lunch line or "I have stories." lands. `variant="quote"` sizes it up for a standalone pull quote; `cite` adds a mono attribution line.

```jsx
<Callout>There is no force stronger than the gravity of free lunch.</Callout>
<Callout variant="quote" cite="Work Life">Flash is dead, long live the API.</Callout>
```

Props: `variant` (`aside` default · `quote`), `cite`.
