# SectionLabel

Mono uppercase kicker that opens a section. Pair it above a serif headline. Add an `index` for the numbered-section feel and `rule` to run a hairline to the container edge.

```jsx
<SectionLabel index="03" rule>The Exciting Stuff</SectionLabel>
<SectionLabel color="muted">What I work with</SectionLabel>
```

Props: `index`, `rule`, `color` (`accent` default · `ink` · `muted`).
