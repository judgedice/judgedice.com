# Rule

Horizontal divider. Plain hairline by default; `weight="ink"` for the editorial black rule, `strong` for the mid-weight. Pass a `label` to sit a mono caption on the line.

```jsx
<Rule />
<Rule weight="ink" />
<Rule label="Career arc" />
<Rule label="Since 2000" align="center" />
```

Props: `label`, `weight` (`hair` · `strong` · `ink`), `align` (`left` · `center` · `right`).
