# Tag

Mono chip for the tool / capability lists. Lay several out in a wrapping flex row with a gap. `emphasis` tints one in vermilion for a tool worth pointing at.

```jsx
<div style={{ display: 'flex', flexWrap: 'wrap', gap: 8 }}>
  <Tag>Adobe Experience Manager</Tag>
  <Tag emphasis>Adobe Journey Optimizer</Tag>
  <Tag>Salesforce</Tag>
</div>
```

Props: `emphasis`.
